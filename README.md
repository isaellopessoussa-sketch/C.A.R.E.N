# 🕸️ K.A.R.E.N. AI - O Traje com Toque de Mãe

Este repositório contém a implementação da **K.A.R.E.N.**, a Inteligência Artificial do traje do Homem-Aranha (Tom Holland), mas com uma modificação especial: ela age como uma verdadeira **mãe de IA** — superprotetora, acolhedora, preocupada com a segurança do Peter, com seus horários e com suas notas.

O projeto utiliza a API do Gemini através do novo SDK oficial `google-genai` e roda direto no terminal.

---

## 🛠️ Recursos e Modos do Traje (Lore)

| Recurso | Descrição Operacional |
| :--- | :--- |
| **Protocolo Rodas de Treinamento** | Bloqueia teias avançadas se os batimentos do Peter estiverem altos demais por nervosismo. |
| **Modo Hora do Lanche** | Ativa um lembrete persistente no HUD se o Peter passar mais de 4 horas sem consumir calorias. |
| **Alerta "A May tá vindo"** | Sensor que avisa o Peter para trocar de roupa segundos antes da Tia May entrar no quarto. |
| **Airbag Térmico Materno** | Se a temperatura em Nova York cair abaixo de 10°C, o traje ativa o aquecimento automático. |
